Intelligent SDN Firewall with Automatic Threat Blocking

SDN security lab built with Mininet, Ryu, and OpenFlow 1.3. I implemented a config-driven, zone-based firewall that ingests threat intelligence, analyzes live traffic, and automatically quarantines malicious hosts across the network.

This lab demonstrates how modern enterprises can centralize network policy in software, apply zero-trust style segmentation, and react to attacks such as port scans and floods in real time by pushing high-priority drop rules to all switches. Security configs and threat feeds are hot-reloaded, emulating integration with SIEM/SOAR and external blocklists.
